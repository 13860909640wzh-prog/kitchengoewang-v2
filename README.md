# 大圣跑酷

一款基于 HTML5 Canvas 的竖屏移动端跑酷小游戏。

## 分类

- 一级分类：Games 游戏项目
- 二级分类：HTML5 Canvas 网页小游戏
- 游戏类型：竖屏跑酷 / 休闲动作
- 技术栈：HTML + CSS + JavaScript + Canvas
- 部署方式：GitHub Pages 静态部署
- 适配方向：移动端竖屏优先

## 项目说明

`kitchengoewang-v2` 当前是一个可直接运行的静态网页游戏项目。根目录下的 `index.html` 包含游戏页面、样式和主要逻辑，图片资源从 `assets/` 目录加载。

## 玩法要点

- 点击屏幕跳跃，支持二段跳
- 吃桃子积累冲刺能量
- 能量满后可触发“筋斗云”冲刺
- 冲刺时吃桃子获得额外分数
- 掉落屏幕底部游戏结束

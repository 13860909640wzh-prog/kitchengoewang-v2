# Reference Library 参考案例库

这个目录用于存储后续创作前可以读取和参考的案例、模板、风格样板和结构范例。

它和 `Templates 模板库` 的区别：

- `Reference Library`：放优秀案例、灵感、结构、风格参考，用来学习和借鉴。
- `Templates`：放可以直接复制复用的空模板、脚手架、版式模板。

## 适合放什么

- 优秀 PPT 案例
- 高级网站案例
- 游戏玩法案例
- UI / 视觉风格参考
- 商业计划书结构参考
- README / 项目文档范例
- 交互体验和动效样板
- 行业案例、竞品分析、参考链接清单

## 建议目录

```text
reference-library/
  ppt-cases/
  website-cases/
  game-cases/
  design-styles/
  business-cases/
  readme-examples/
  interaction-patterns/
  links.md
```

## 使用方式

以后可以这样要求：

```text
做 PPT 前先读参考案例库
```

```text
把这个网站案例放到 Reference Library
```

```text
参考 game-cases 里的玩法，帮我生成一个新小游戏
```

## 记录模板

每个案例建议按这个格式记录：

```markdown
# 案例名称

## 来源

链接或文件位置。

## 分类

- 大类：
- 子类：
- 适合参考：

## 亮点

- 亮点 1
- 亮点 2
- 亮点 3

## 可复用结构

说明这个案例里哪些结构、版式、文案、交互或视觉风格值得复用。
```
